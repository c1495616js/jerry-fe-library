## User List

![](img/2022-08-05-14-33-03.png)

### Scenario

- `odd` align left
- `even` align right.

### CSS solutions

![](img/2022-08-05-14-35-55.png)

```css
&:nth-child(odd) {
  aligh-self: flex-start;
}
&:nth-child(even) {
  aligh-self: flex-end;
}
```

To be responsive

```css
@media (min-width: 460px) {
  &:nth-child(odd) {
    aligh-self: flex-start;
  }
  &:nth-child(even) {
    aligh-self: flex-end;
  }
}
```
